# T13.2.1: 中間結果永続化・入力ハッシュスキップ

- **ステータス**: Todo
- **優先度**: Must
- **サイズ**: M
- **対応要件**: R13 / F13.2

## 目的
各ステップの中間結果を永続化し、入力が変わらない限り再実行をスキップする

## 完了条件
- [ ] 各ステップの出力が `.groa/{stepName}.json` に保存される
- [ ] 入力データのハッシュが計算され、キャッシュファイルに記録される
- [ ] 再実行時、入力ハッシュが一致すればステップをスキップする
- [ ] --force フラグでスキップを無視して再実行できる
- [ ] `groa clean` で特定ステップまたは全キャッシュを削除できる
- [ ] APIキーがキャッシュJSONに書き出されない
- [ ] ユニットテストでハッシュ計算・スキップ判定・削除が検証されている

## 依存関係
- 先行: T2.1.1
- 後続: T13.1.1, T14.1.4

## 実装メモ
- spec.md §7.1 に準拠
- ハッシュは入力データ + 設定値から計算
- cacheDir は設定で変更可能（デフォルト ".groa"）
