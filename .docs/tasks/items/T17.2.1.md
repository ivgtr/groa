## 実装メモ
- Anthropic API とはリクエスト/レスポンス形式が根本的に異なるため、共通基底クラスは持たない
- `LlmBackend` インターフェースのみを共通とし、`withRetry` ユーティリティを共有する
- `useCache` / `useBatch` は OpenRouter 非対応のため無視する
- `cachedTokens` は常に `0` を返す
