# T1.1.1: pnpm workspace・ビルド・テスト・lint基盤セットアップ

- **ステータス**: Todo
- **優先度**: Must
- **サイズ**: M
- **対応要件**: R1 / F1.1

## 目的
全パッケージが共通のビルド・テスト・lint設定を利用できるモノレポ基盤を構築する

## 完了条件
- [ ] pnpm-workspace.yaml が存在し、packages/* を対象としている
- [ ] ルートの tsconfig.json (base) が strict: true で定義されている
- [ ] 各パッケージの tsconfig.json が base を extends している
- [ ] vitest.config.ts がルートに存在し、--project でパッケージ単位実行が可能
- [ ] ESLint設定が存在し、import/no-cycle ルールが有効
- [ ] `pnpm build` でルートからの一括ビルドが成功する
- [ ] `pnpm test` でルートからの一括テスト実行が成功する（テストが0件でもエラーにならない）
- [ ] `pnpm lint` でルートからの一括lint実行が成功する

## 依存関係
- 先行: なし
- 後続: T1.2.1, T2.1.1, 他全パッケージ

## 実装メモ
- TypeScript 5.x + pnpm workspace + Vitest + ESLint
- パッケージ間のビルド順序は pnpm の workspace protocol で自動解決
- tsconfig の paths 設定で開発時の型解決を高速化
