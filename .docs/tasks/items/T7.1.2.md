- **ステータス**: Todo

## 目的
各クラスタに対してSonnetモデルで人格特徴を抽出するLLM分析を実装する

## 完了条件
- [ ] クラスタ固有の統計サブセット・語尾パターン・頻出表現をコンテキストに含むプロンプトが構築される
- [ ] portrait（500-1500字）が出力される
- [ ] representativeTweets（最大10件、選定理由付き）が出力される
- [ ] attitudePatterns（3-5件、名前・説明・実例ID）が出力される
- [ ] temperature 0.0 が指定されている
- [ ] Zodバリデーション失敗時、当該クラスタをスキップしログに警告する
- [ ] ClusterAnalysis[] が最終出力される
- [ ] ユニットテスト（モック）でプロンプト構築とレスポンスパースが検証されている
